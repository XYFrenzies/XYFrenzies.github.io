PositiviTea

As part of the 1st year major production at AIE, PositiviTea was created by 1 other programmer, 2 designers and 2 artists.

This project was created using C# and Unity Game engine.

Cops and Robbers with AI

As part of the 1st years developement of creating a game with AI, Cops and Robbers is based off a robber finding as many coins as possible before getting caught by the cop.

The AI techniques that used were: A* pathfinding, wander, follow, flee and seek behaviours.

This project was created using C++ openGL.

Archery Madness VR

When researching into virtual reality for the year 2 project, Archery Madness was developed to give the feel of a torn down carnival that enables the player to shoot targets with a bow and arrow.

This project was created using C# in Unity Engine.
